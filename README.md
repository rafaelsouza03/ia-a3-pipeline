# Análise de Consumo de Energia na Holanda (2009-2020)

Este repositório contém um notebook de análise de dados que explora o consumo energético na Holanda ao longo do período entre 2009 e 2020. Utiliza técnicas de tratamento, exploração e visualização de dados para revelar tendências, padrões e insights relevantes acerca do consumo energético ao longo dos anos nesse país europeu.

## Arquivo principal

- [`a3_pipeline.ipynb`](https://github.com/rafaelsouza03/ia-a3-pipeline/blob/main/a3_pipeline.ipynb): Notebook Jupyter contendo todas as etapas do pipeline de análise, desde o carregamento dos dados até as visualizações finais e conclusões.

## Estrutura do notebook

O notebook aborda os seguintes tópicos:

- **Introdução**  
  Contextualização do problema e objetivo da análise.

- **Importação de bibliotecas e dados**  
  Carregamento das bibliotecas essenciais de Python para análise de dados (Pandas, Numpy, Matplotlib, Seaborn, etc.), bem como os conjuntos de dados utilizados.

- **Exploração inicial dos dados**  
  Visualização das primeiras linhas do dataset, análise do formato, tipos de dados e informações básicas.

- **Tratamento de dados**  
  Processos para corrigir inconsistências, tratar valores ausentes, remover outliers e preparar os dados para análise.

- **Análise exploratória**  
  Visualização de distribuições, padrões temporais, correlações e outros aspectos relevantes para entender o consumo de energia ao longo do tempo.

- **Conclusões**  
  Sumário dos principais achados da análise, com insights e possíveis recomendações.

## Como executar o notebook

1. Clone este repositório:
   ```bash
   git clone https://github.com/rafaelsouza03/ia-a3-pipeline.git
   ```
2. Certifique-se de ter o Python 3 e o Jupyter Notebook instalados.
3. Instale as bibliotecas necessárias:
   ```bash
   pip install -r requirements.txt
   ```
   (Se não houver um arquivo `requirements.txt`, instale manualmente pandas, numpy, matplotlib, seaborn e qualquer outra biblioteca necessária.)
4. Inicie o Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
5. Abra o arquivo `a3_pipeline.ipynb` e siga as células para reproduzir a análise.
